# Репозиторий My_ScriptsApp

### Этот репозиторий будет для разных <u>простых</u> windows или linux скриптов.

### Список моих скриптов:
1. WebToPDF_Converter
2. Win_timer_off
3. adb_Connect_android_win&&adb
4. win_AntiSleep_dispay
5. win_Search_word_in_docx

### win - обозначение Windows, bsh - обозначение Linux
Если в начале файле стоят такие обозначения

### Доделать_добавить_пофиксить
Также доделать скрипт с конвертом web_страницы в pdf_файл, чтобы он был универсальный 

Также скомпилированные в .exe-файл (для win) и все остальные скрипты

Какие - то прям проекты будут как отдельная директория.
